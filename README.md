React Accordion App

A simple and interactive Accordion application built using React.js. This project allows users to expand and collapse questions to view their answers. It also includes a Multi-Selection mode where multiple accordion items can remain open simultaneously.

🚀 Features
Expand and collapse accordion items
Display questions and answers dynamically
Enable/Disable Multi-Selection mode
Clean and responsive UI
State management using React Hooks (useState)
Beginner-friendly React project
🛠️ Technologies Used
React.js
JavaScript (ES6+)
CSS3
HTML5
📸 Project Preview

The application contains:

FAQ-style accordion sections
Single Selection Mode (only one answer visible at a time)
Multi Selection Mode (multiple answers visible simultaneously)
Toggle button to switch between modes
